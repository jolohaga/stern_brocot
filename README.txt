= SternBrocot

* http://rubygems.org/gems/stern_brocot

== Description:

Generate Stern-Brocot fractions, trees and series.

== Features/Problems:

* Requires Ruby 1.9+

== Synopsis:

=== Using Fraction

See SternBrocot::Fraction
  
=== Using Tree

See SternBrocot::Tree

=== Using Series

See SternBrocot::Series
  
== Requirements:

* Ruby 1.9+

== Install:

sudo gem install stern_brocot

== Developers:

* Ruby 1.9+
* RSpec required to run tests.
* Echoe required to run rake tasks.
